Reflections

Before taking the CIS 241 class, I have just a little experience in creating a Website on WordPress. Coding in HTML5 and CSS is an unfamiliar term to me. I thought CIS 241 would be difficult for me, who didn't have any foundation in coding. However, the study material on CIW is easy to understand for me. After ten lessons in the book, I learned about the basics of website development such as HTML5, CSS, and Internet business or marketing concepts.
Besides, I can find the errors as well as how to fix them after this class, which I thought that I couldn't understand before. The thing that attracts my interest and curiosity the most is CSS, an external file that can help web developers changing the styles of HTML elements. Instead of changing every webpage on the site, I only need to change a small part of the CSS file and all the webpage's style change. In the future, I want to take more Computer Information System classes and improve my coding skills.
